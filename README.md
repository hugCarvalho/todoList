# TODO APP

## Project Specifications

- Built with React, styled with SASS

#### React specifics:

- Ref forwarding / Modal / PropTypes

## App functionalities

- Add / Edit / Delete a todo
- Search for a todo
- Delete All todos
- Check / uncheck All
- Show / Hide completed
- Save to local storage
